	ii. $\int{cos(\sqrt(x))}dx$ 
		\begin{align}  
		\int {cos(\sqrt{x)}} dx 
		&= 2 \int{\sqrt{x} \ cos(\sqrt{x})} \ d\sqrt{x} \\
		&= 2 \sqrt{x} \ sin(\sqrt{x}) + 2 cos(\sqrt{x}) + C 
	\end{align}
	
	\[

\\\bottomrule
